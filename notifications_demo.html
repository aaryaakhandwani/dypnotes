Notification System

This is a demonstration of the modern notification system designed for PHP applications. It features clean SVGs, smooth layout transitions, and simple HTML/CSS/JS without any external frameworks.

// Example Usage:
NotificationSystem.show([
    { 
        title: 'Welcome!', 
        message: 'You have logged in successfully.', 
        type: 'info', 
        duration: 4000 
    }
]);
